# Influencer Advertising Analysis

A data-driven web application analyzing social media influencer campaigns across platforms. Built with Python & Flask, this project helps brands understand engagement patterns, classify creators, and make advertising decisions.

## Key Metrics
- Posts analyzed: 200
- Influencers: 110
- Platforms: 6
- Avg engagement: 4.42%
- Sponsored rate: 31.5%
- Top platform: Instagram
- Top audience country: US

## Features
- K-Means clustering of influencers into 3 groups
- Visual insights: engagement by platform, followers vs engagement, post type distribution
- Summary dashboards with actionable recommendations

## Tech Stack
Python, Flask, Pandas, Seaborn, Matplotlib, Scikit-learn
